Getting Started

Welcome to Lens - your comprehensive personal finance companion! This Progressive Web App (PWA) provides powerful expense tracking, goal management, and financial optimization tools.

First Steps:

  1. Complete the onboarding tour to set up your income, goals, and savings
  2. Visit to configure your monthly budget and preferences
  3. Connect bank accounts via Plaid for automatic transaction import
  4. Start tracking expenses and watch your financial insights grow

Key Benefits: Real-time goal impact analysis, credit card points optimization, smart spending forecasts, and YNAB-style savings allocation.

Dashboard Tabs Overview

Overview

Your financial command center with live snapshot of monthly performance.

  • Key Financial Stats: Monthly budget, spending total, remaining budget, and income breakdown (click income card for details)
  • AI-Powered Projections: Visual budget progress bar with machine learning forecasts for month-end spending
  • Budget Health Score (0-100): Comprehensive financial wellness metric with detailed breakdown tooltip
  • Essential vs Non-Essential Spending: Categorized spending analysis for better budget awareness
  • Goal Progress Cards: Real-time timeline updates showing how current spending affects goal completion dates
  • Quick Action Buttons: One-click expense and recurring item creation

Expenses

Comprehensive expense management with intelligent categorization and bulk operations.

  • Smart Auto-Categorization: AI-powered category suggestions based on merchant names and spending patterns
  • Advanced Filtering: Date ranges, categories, expense types, and auto-generated recurring items
  • Bulk Operations: Multi-select with checkboxes for efficient batch deletions
  • Real-time Impact: Instant goal timeline updates when expenses are added/modified
  • Import Integration: Seamless merging with bank-imported transactions (no duplicates)

Recurring

Automated income and expense tracking for predictable financial planning.

  • Dual Mode Support: Income (salaries, freelance) and expenses (subscriptions, bills)
  • Automatic Processing: Scheduled creation of expense records on due dates
  • Dynamic Income Integration: Recurring income automatically updates monthly totals
  • Flexible Status Management: Pause (cancelled) and resume (active) recurring items
  • Forecast Integration: Upcoming recurring items appear in 7-day forecast analysis

Insights

Advanced AI analytics with banking integration and optimization recommendations.

  • Credit Card Points Optimization: Identifies missed rewards from imported transactions with annual value calculations
  • Real-time Goal Impact Analysis: Shows how spending affects completion timelines with optimization scenarios
  • Smart Actionable Recommendations: Dismissible cards with specific tips for points optimization and budget improvements
  • 7-Day Forecast (2-column grid): Upcoming expenses with High/Medium/Low confidence levels
  • Month-End Projections: ML-powered forecasts for spending, savings, and goal progress
  • Monthly Story Narrative: Personalized summary highlighting achievements and key metrics

Goals & Real-time Impact Analysis

Goal System Overview

YNAB-style "give every dollar a job" approach with real-time timeline calculations and optimization impact analysis.

Goal Types & Templates

  • House Deposit: Property purchase savings with market-based targets
  • Emergency Fund: 3-6 months expense coverage calculation
  • Dream Vacation: Travel and experience savings planning
  • Investment Fund: Long-term wealth building goals
  • Debt Payoff: Strategic debt elimination with 20% income allocation
  • Custom Goals: Flexible targets for any financial objective

Advanced Features

  • Real-time Timeline Updates: Instant recalculation when transactions change
  • Optimization Comparison: "Current" vs "With Optimization" scenarios
  • Smart Savings Estimation: Realistic capacity based on income and spending
  • Progress Acceleration Metrics: Days/months saved through optimization
  • Cross-tab Synchronization: Consistent progress display across Overview and Insights

Savings Allocation System

  • Total Savings Pool: Complete savings across all accounts
  • Goal Allocation: Assign specific amounts to individual goals
  • Unallocated Tracking: Monitor unassigned savings available for new goals
  • Multiple Goal Support: Split savings across various objectives
  • Net Worth Calculation: Automatic debt consideration for accurate financial position

Banking Integration & Transaction Import

Plaid Integration Setup

  1. Ensure backend server (Node.js/Express) is running on localhost:8000
  2. Configure Plaid API credentials in server environment
  3. Click "Connect Bank Account" in Settings to start secure authentication
  4. Select your bank and complete OAuth flow

Import Features

  • 30-Day Transaction History: Automatic import of recent banking activity
  • Smart Categorization: Merchant-based category assignment
  • Duplicate Prevention: Intelligent deduplication with manual entries
  • Real-time Processing: Immediate goal timeline updates

Security & Privacy

  • Bank-Grade Security: Plaid-powered encryption and authentication
  • No Credential Storage: Only secure access tokens retained
  • Local Data Storage: Financial data stored in browser localStorage
  • Token-based API: Secure backend communication without exposing credentials

Points Optimization Analysis

Imported transactions are automatically analyzed for credit card rewards opportunities, showing potential annual value and optimal card recommendations for each spending category.

Points Optimization Calculations

Credit Card Database

Amex Gold: 4x Food & Drink, 1x Others
Amex Green: 3x Transport, 1x Others
Virgin Red: 3x Shopping, 2x Transport
Chase Sapphire: 2x Travel & Dining

Point Valuations (Conservative)

Amex Points: 0.005 per point
Virgin Points: 0.01 per point
ANA Miles: 0.012 per mile
Chase Points: 0.008 per point

Optimization Formula

potentialValue = transactionAmount pointRate pointValue
annualSavings = monthlyMissedValue 12
optimizationImpact = annualSavings 12 (monthly boost)
Example: 100 at Tesco with Amex Gold
100 4 0.005 = 2.00 potential value

Goal Timeline Calculations

Debt Payoff Goals

availableIncome = max(0, monthlyIncome - monthlyBudget)
paymentCapacity = availableIncome 0.2
monthlyPayment = max(50, paymentCapacity)
monthsToCompletion = remainingDebt monthlyPayment

Allocates 20% of available income to debt payments with 50 minimum.

Savings Goals

surplus = max(0, monthlyIncome - monthlySpending)
monthlySavings = max(50, min(surplus 0.5, monthlyIncome 0.1))
monthsToGoal = remainingAmount monthlySavings

Conservative approach using 50% of surplus or 10% of income, whichever is lower.

Optimization Impact Calculation

monthlyPointsValue = (monthlySpending avgPointsRate avgPointValue)
optimizedSavings = baseSavings + monthlyPointsValue
optimizedTimeline = remainingAmount optimizedSavings
timeSaved = originalTimeline - optimizedTimeline

Shows how credit card optimization accelerates goal completion.

Smart Estimation Features

  • Realistic Savings Rates: Accounts for actual spending patterns and income volatility
  • Goal-Specific Logic: Different calculation methods for debt vs savings goals
  • Minimum Thresholds: Ensures meaningful monthly progress (50 minimum)
  • Income Protection: Never allocates more than 10% of total income to single goal

Goal Modal Calculations

Understanding Your Financial Position

Core Metrics:
  • Goal Target: Total amount you're saving for
  • Total Savings: Complete savings across all accounts
  • Allocated Amount: Portion assigned to this specific goal
  • Still Needed: Gap between target and allocated amount
Advanced Calculations:
  • Unallocated Savings: Available for new goal assignments
  • Net Worth: Total Savings - Total Debt
  • Allocation Percentage: Goal allocation as % of total savings
  • Progress Percentage: (Allocated Target) 100

YNAB-Style Allocation System

Core Principle: "Give every dollar a job" - Every pound of savings should be allocated to a specific goal or purpose.

totalSavings = 10,000
allocatedToGoals = 8,500
unallocated = 1,500
netWorth = 10,000 - 2,000 (debt) = 8,000
  • Multiple Goal Support: Split total savings across various objectives
  • Flexible Reallocation: Move money between goals as priorities change
  • Unallocated Buffer: Keep some savings unassigned for new opportunities
  • Debt Integration: Net worth calculation factors in total debt automatically

Goal Editing & Management

  • Real-time Updates: Changes instantly reflect across all tabs and dashboards
  • Validation Logic: Prevents over-allocation beyond available savings
  • Timeline Recalculation: Automatic timeline updates when allocations change
  • Cross-tab Synchronization: Consistent data display in Overview and Insights tabs

Settings & Data Management

Banking Configuration

  • Plaid Integration: Secure bank account connections
  • Backend Server: Node.js/Express on localhost:8000
  • API Credentials: Plaid sandbox/development keys required
  • Token Management: Secure access token storage and refresh

Personalization

  • Currency Support: Multiple currency display options
  • Theme Toggle: Light/dark mode with system preference detection
  • Notification Settings: Customizable alerts and reminders
  • Display Preferences: Chart types, dashboard layout options

Data Management & Privacy

Local Storage:
  • All financial data stored in browser localStorage
  • No server-side personal data storage
  • Automatic data persistence across sessions
  • Clear browser data will erase all records
Backup & Recovery:
  • JSON Export: Complete data backup capability
  • Import Restoration: Restore from previous exports
  • Regular Exports: Recommended weekly backup schedule
  • Version Control: Date-stamped export files

Security & Privacy Considerations

  • Bank Credentials: Never stored locally - only secure Plaid tokens
  • Data Encryption: Banking tokens encrypted in transit
  • Local-Only Storage: Personal financial data remains on your device
  • No Analytics: No third-party tracking or data collection
  • PWA Security: Service worker caching with secure resource handling

Technical Requirements

Frontend (PWA)

  • Modern Browser: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
  • JavaScript: ES6+ features required
  • Local Storage: Minimum 10MB available space
  • PWA Support: Service workers and manifest support

Backend (Optional Banking)

  • Node.js: Version 16+ with Express framework
  • Plaid API: Sandbox or development credentials
  • CORS Configuration: Localhost frontend access
  • Environment Variables: Secure API key management
